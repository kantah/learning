wget ссылка  # скачать файл по ссылке в текущую директорию
wget -P путь_до_директории ссылка  # скачать файл по ссылке и сохранить в указанной директории
wget -O путь_до_файла ссылка  # скачать файл по ссылке и сохранить в указанной директории под указанным именем
wget -c ссылка - докачать файл по ссылке(например при обрыве связи). Если не использовать -с, загрузка начнется сначала
wget --spider - проверить доступность файла по ссылке
wget -i текстовый_файл - скачать несколько файлов по ссылкам из текстового файла
wget --limit-rate=число - ограничение скорости загрузки. По умолчанию - байты, К - килобайты, М - мегабайты
wget -b - фоновая загрузка
wget --tries=число - указание числа попыток загрузки файла
wget FTP-URL - загрузка с анонимного ftp сервера
wget --ftp-user=user --ftp-password=password FTP-URL - загрузка с ftp сервера с авторизацией
wget --mirror -p --convert-links -P путь_до_папки ссылка - создание локальной копии сайта. -p - производится загрузка всех файлов, необходимых для отображения html страниц; --convert-links - ссылки в
документах будут преобразованы для локального просмотра; -Р путь_до_папки - материалы будут сохранены в указанную папку
wget --reject=тип ссылка - пропуск файлов определенного типа
wget -o файл ссылка - загрузка с использованием собственного лог-файла
