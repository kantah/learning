cat файл  # вывести содержимое файла на экран
less файл  # открыть файл на чтение. q - выход, / - поиск, g - в начало, G - в конец
nano файл  # редактировать файл. ctrl+x - выход

программа<файл  # брать stdin из файла
программа>файл  # выводить stdout в файл
программа>>файл  # выводить stdout в файл с дозаписью
программа 2>файл  # выводить stderr в файл
программа 2>>файл  # выводить stderr в файл с дозаписью
программа &>out.txt  # выводить stderr и stdout в один файл

Конвейер pipe
прогр1 | прогр2 | прогр3 |...| прогрN  # передать stdout прогр1 в качестве stdin прогр2, stdout прогр2 - stdin прогр3 и т.д.
cat names | ./interacter.py | less  # cat выводит в stdout имена из файла names< которые поступают в stdin программы interacter. interacter их обработает и выведет в stdout, который поступит в 
качестве stdin программе less
