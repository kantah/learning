Вход:
im <файл>
vim <файл1 файл2>

Главные команды:
:q - выход
:help - справка

vimtutor - туториал

Перемещение:
    По символам:
	стрелки
	h, j, k, l
    По словам:
        w, W - в начало следующего слова
	e, E - в конец
	b, B - в начало предыдущего
    По строке:
	0 - начало строки
	^ - первый непробельный символ
	$ - конец строки
    По файлу:
	gg - начало файла
	G - конец файла
	:<номер>, Enter - перейти к строке <номер>
	ctrl+d, ctrl+u - вниз/вверх на N строк, где N обычно полэкрана

Удаление:
    x - удаление символа под курсором
    X - символ до курсора

    Сразу много:
        d<кол-во><ход>
	de - удалить до конца текущего слова
	d$ - удалить до конца строки
	d5w - удалить 5 слов
	
    Специальная команда:
	dd - удалить строку
	d<число>d - удалить <число> строк

Создание:
    i - войти в insertion mode
    a - сдвинуть курсор вправо и войти в insertion mode
    I == ^i
    A == $a
    o - вставить пустую строку под курсором и редактировать ее
    O - вставить пустую строку над курсором и редактировать ее

Копирование:
	y<кол-во><ход>
	yy - скопировать строку

Вставка:
	p - вставить после
	P - вставить перед

Поиск:
	/<текст> - искать текст вперед
	/? - искать назад
	n - искать следующее вхождение
	N - искать предыдущее вхождение

Замена:
	:%s/<что ищем>/<на что меняем>/<флаги>
		g - много замен в строке
		c - подтверждение

Отмена/возврат:
	u - отменить последнее действие
	ctrl+r - вернуть отмененное действие

Сохранение работы:
	:w - сохранить изменения
	:wq - сохранить и выйти
	:q! - выйти без сохранения
	:w <файл> - сохранить изменения в <файл>

Для изменения цветовой схемы редактора VIM, надо открыть файл vimrc для редактирования.
Например так: sudo vim ~/vimrc
В этот файл добавить строчку: colorscheme desert .
Сохранить и выйти. Цветовая схема будет более удобна чем стоящая по дефолту.
На самом деле,цветовых схем несколько, для неискушенных, я лично, советую   desert .
Узнать текущую цветовую схему можно в самом редакторе , командой: :colorscheme .
