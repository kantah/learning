ssh логин@адрес_сервера -p порт - команда входа

Авторизация с использованием ключей:
ssh-keygen - создание ключа
ssh-add - сообщить системе о ключе
cat ~/.ssh/.id_rsa.pub - просмотр публичного ключа
nano ~/.ssh/authorized_keys - редактирование авторизованных ключей на сервере
ssh-copy-id -i ~/.ssh/id_rsa.pub myuser@server.domain.com - еще один способ добавления ключа на сервер
