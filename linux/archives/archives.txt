zip архив.zip файл1, файл2... - запаковать перечисленные файлы и/или папки в архив.zip
gzip файл - запаковать файл в файл.gz, исходный файл удалить. При архивировании исходный файл удаляется. Нельзя запаковать в архив несколько файлов или папку.
tar -cvf архив.tar файл1, файл2 ... - упаковать перечисленные файлы и/или папки в архив.tar(без сжатия). -с - обязательная опция, отвечает за создание архива; -v - выводит информацию о ходе выполнения
    команды на экран; -f - отвечает за то, что создаваться будет именно файл на файловой системе. Раньше команда tar работала с магнитными лентами. Опция должна идти обязательно в конце. Нет свободного 
    порядка опций
gzip архив.tar - запаковать архив.tar в архив.tar.gz, исходный архив.tar удалить
unzip архив.zip - распаковать содержимое архива.zip
gunzip архив.gz - распаковать содержимое архива.gz, файл архив.gz удалить
tar -czvf архив.tar.gz файл1 файл2... - запаковать перечисленные файлы в архив.tar.gz с сжатием gzip
tar -xvf aрхив.tar - распаковать архив tar
tar -xzvf архив.tar.gz - распаковать архив.tar.gz

Архиватор bzip
bzip2 файл
bunzip2 файл.bz2
tar -cjvf файл1 файл2...
tar -xjvf архив.tar.bz2
